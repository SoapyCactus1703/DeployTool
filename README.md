# 西林民高部署工具

专为西林民高教学终端自动化部署设计的工具软件。

## 功能特性

- **流程部署** - 教学部署、办公部署一键完成
- **系统优化** - 优化套餐、磁盘与性能、系统服务、时间设置
- **软件与驱动** - 驱动安装、教学软件、办公软件
- **网络与用户** - 网络设置、用户管理
- **系统管理** - 激活系统、Windows轻松设置等

## 系统要求

- Windows 10/11 64位
- 需要管理员权限运行

## 开发环境

- .NET 8.0
- WPF (Windows Presentation Foundation)

## 构建项目

```bash
cd DeployTool/部署工具/DeployTool
dotnet restore
dotnet build
dotnet publish -c Release
```
